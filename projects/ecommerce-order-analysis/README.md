# 电商订单数据分析

## 项目简介
本项目使用和鲸社区提供的电商订单数据集，对5.2万条订单记录进行清洗、分析和可视化，探索订单量趋势、订单状态分布、配送时效等业务指标。

## 数据集来源
- 和鲸社区 - 电商订单数据集
- 包含52145条订单记录

## 分析工具
- Excel（数据清洗、数据透视表、可视化图表）

## 分析内容

### 1. 数据清洗
- 将日期序列号转换为标准日期格式
- 检查并处理空值、重复值
- 提取下单日期字段，计算配送天数

### 2. 每日订单量趋势
- 使用数据透视表按日统计订单量
- 绘制折线图发现周末订单量高于工作日
- 建议：周末可增加营销活动

### 3. 订单状态分布
- 统计各订单状态占比
- 绘制饼图发现“已送达”订单占绝大多数
- 数据质量良好

### 4. 配送时效分析
- 计算每个订单的配送天数
- 按3天一组分组统计
- 大部分订单在10天以上送达
