# 🚀 数据分析实战项目

这里展示我用 SQL 解决实际业务问题的完整项目，体现从数据清洗到业务洞察的全流程能力。

## 📊 项目列表

### 1. 电商销售分析系统（筹备中）
- **业务场景**：模拟电商平台销售数据分析
- **技术栈**：MySQL，多表连接，窗口函数，RFM分析
- **产出**：销售报表，用户分群，商品推荐策略

### 2. 用户行为分析平台（规划中）
- **业务场景**：用户点击流与转化分析
- **技术栈**：事件数据建模，漏斗分析，留存计算

## 🛠️ 项目方法论
每个项目遵循：
1. **业务理解**：明确分析目标与指标
2. **数据准备**：清洗、转换、建模
3. **SQL分析**：编写可复用的分析脚本
4. **报告输出**：可视化与业务建议
